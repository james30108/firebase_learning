Firebase Web-Hosting

ทดสอบการอัปโหลดเว็บ

ขั้นตอนที่ 1 ลง npm install -g firebase-tools ขั้นตอนที่ 2 login เข้าสู่ระบบจากในเครื่อง โดยใช้คำสั่ง firebase login ขั้นตอนที่ 3 เริ่มต้นสร้างโฮสติ้งโดยใช้คำสั่ง firebase login ขั้นตอนที่ 4 ลากไฟล์ที่ต้องการเข้าไปใส่ใน folder ชื่อ public ขั้นตอนที่ 5 เมื่อทำทุกอย่างเสร้จสิ้นให้ทำการอัปเดตเว็บโดยใช้คำสั่ง firebase deploy โดยจะใช้คำสั่งนี้ทุกครั้งที่มีการอัปเดตเว็บ
